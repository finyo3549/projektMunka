## Installation

 1.  Repository klónozása:

     `git clone https://github.com/username/project-name.git`


2. Composer telepítés
   
   `composer install`

   
3. Environment másolása:
   
   `cp .env.example .env`

4. Artisan telepítés:

   `php artisan install`

5. App key generálása:

   `php artisan generate:key`

6. Fejlesztői szerver indítása:

   `php artisan serve`
